Simon Game Rules

Welcome to the Simon Game! Test your memory and see how many sequences you can remember.

Objective: Repeat the sequence of colors as long as you can.

How to Play:

1. The game will start with a single color being highlighted. Remember the color.

2. Click the same color button to repeat the sequence.

3. The game will then add one more color to the end.

4. Repeat the entire sequence by clicking the correct color buttons in the correct order.

5. Each time you successfully repeat the sequence, the game adds one more color to the sequence.

6. If you click the wrong color, the game is over, and you'll need to start again.

Try to achieve the highest level possible by correctly repeating the sequences!

Press the start button start the game and enjoy!
